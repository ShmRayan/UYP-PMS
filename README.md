## Project Description
This project is about the system **UYP-PMS (Umelaphi Yaodian Pharmacy Prescription Management System)**.
It is a pharmacy prescription management system, developed as part of the SEG3502 course at the University of Ottawa.

The repository includes:
- The **domain substructure** following Domain-Driven Design (DDD).
- The **application commands**, defined as Java interfaces.
- The **initial contracts**, described using Cucumber `.feature` files.
